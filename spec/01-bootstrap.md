# Step 1: Project Bootstrap

## Goal
Set up the repository structure and baseline tooling so the frontend and backend can be built independently.

## What to build
- Create a `frontend/` app using React + TypeScript.
- Create a `backend/` app using Node.js + Express + TypeScript.
- Add shared repo files like `.gitignore`, `README.md`, and example environment files.
- Add root-level scripts or documented commands for running each app.

## What to avoid
- Do not implement URL shortening logic yet.
- Do not connect to MongoDB yet unless it is only for local connection setup.
- Do not spend time on UI styling at this stage.

## Done when
- Both apps can be started locally.
- The repo has a clear folder structure.
- Someone reading the repo understands where frontend and backend code will live.

## Suggested commit
- `chore: added project structure`
