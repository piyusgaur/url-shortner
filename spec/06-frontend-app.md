# Step 6: Frontend App

## Goal
Build the React UI for shortening URLs and showing results.

## What to build
- Create a form for entering a long URL.
- Allow optional custom alias input if supported by the API.
- Show the generated short URL after success.
- Show loading and error states.
- Keep the interface simple and easy to scan.

## What to avoid
- Do not overbuild a dashboard unless it adds clear value.
- Do not couple the UI to backend internals.

## Done when
- A user can submit a URL from the browser.
- The UI displays success and error feedback.
- The short link is visible and usable after submission.

## Suggested commit
- `feat(frontend): add shorten url interface`
