# url-shortner

URL shortener take-home project scaffold.

## Current Structure
- `frontend/` React + TypeScript app
- `backend/` Node.js + Express + TypeScript API
- `spec/` step-by-step implementation specs and implementation notes

## Planned Commands
- `npm run dev:backend`
- `npm run dev:frontend`
- `npm run test`

The project is being built in small steps so each commit stays easy to review.
